동작 방식

booking-data.json에서 오늘 + 예매 오픈 전 경기를 읽어 자동 버튼을 만듭니다. 경기 버튼을 선택하면 스케줄 페이지 URL이 TARGET_URL에 연결되고(NetFunnel 정상 통과 경로), 직접 예매 URL은 보조 버튼으로 별도 노출됩니다. TARGET_TIME과 정밀 시계가 일치하거나 지나면 메인 예매하기 버튼의 disabled가 즉시 제거되고, 클릭 시 같은 탭에서 스케줄 페이지로 이동합니다. 도착 후 해당 경기의 예매 버튼을 한 번 더 누르면 NetFunnel 키와 함께 정상 예매 페이지로 진입합니다.

GitHub Actions가 매일 09:00 KST에 booking-data.json을 갱신합니다. 티켓링크 구조가 바뀌면 스크래퍼 선택 로직을 조정해야 할 수 있습니다.
