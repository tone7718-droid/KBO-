구현 방식

화면 갱신은 requestAnimationFrame으로 처리하고, 실제 표시 시각은 performance.now()에 Date.now() 기반 보정값을 더해 계산합니다. 그래서 브라우저 렌더링 타이밍은 부드럽게 유지하면서도 밀리초 단위 표시 흔들림을 줄입니다.

표시 정밀도는 1ms 단위입니다. 실제 절대 시각 정확도는 사용 중인 기기의 시스템 시계 동기화 상태에 영향을 받습니다.
