구현 방식

fetch로 세계 표준시 API 또는 서버 응답의 Date 헤더를 읽고, 요청 시작·응답 완료의 중간 지점을 기준으로 서버와 로컬 시계의 오차를 계산합니다. 표시 시각은 performance.now() + 로컬 보정값 + 서버 Offset으로 계산하고, 화면 갱신은 requestAnimationFrame으로 처리합니다.

자동 새 창 열기는 브라우저 팝업 차단 정책에 의해 막힐 수 있습니다. 막히면 활성화된 “예매 시작” 버튼을 직접 누르면 됩니다.
