# KBO 예매 보조 PWA

매크로 프로그램이 아닌 **예매 시간을 줄일 수 있는 보조 프로그램**입니다.

이 프로젝트는 KBO 경기의 예매 오픈 시각을 놓치지 않도록 돕는 정적 PWA입니다. 홈팀 기준 예매 규칙을 읽어 관심 경기의 예매 오픈 예상 시각을 계산하고, 예매처 바로가기, 좌석 선호 메모, 캘린더 알림 파일, 브라우저 알림 예약을 제공합니다.

## 핵심 원칙

이 앱은 다음 기능을 의도적으로 제공하지 않습니다.

- CAPTCHA 우회
- 대기열 우회
- 좌석 자동 클릭
- 자동 결제
- 티켓링크 페이지 반복 새로고침
- 다계정/프록시 접속

목표는 예매처 시스템을 우회하는 것이 아니라, 사용자가 정시에 직접 예매할 수 있도록 준비 시간을 줄이는 것입니다.

## 현재 구현된 기능

- 관심 경기 직접 추가
- 홈팀 기준 예매 오픈 예상 시각 계산
- 삼성 라이온즈 기본 예매 링크: `https://m.ticketlink.co.kr/sports/137/57`
- 예매 오픈 전 버튼 잠금
- 예매 오픈 시각 이후 `예매 바로가기` 버튼 자동 활성화
- 앱 내부 1초 단위 카운트다운 갱신
- 구단별 예매 규칙 파일 분리: `data/ticket_rules.json`
- 인원/좌석 선호 메모 저장
- 준비 메모 클립보드 복사
- `.ics` 캘린더 알림 파일 생성: 30분 전, 10분 전, 5분 전, 1분 전
- 브라우저 로컬 알림 예약: 30분 전, 10분 전, 5분 전, 1분 전
- PWA manifest 및 service worker 기본 구성
- 모바일 중심 반응형 UI

## 중요한 제한 사항

`data/ticket_rules.json`의 예매 오픈일과 오픈 시간은 기본적으로 **추정값**입니다. 삼성 라이온즈 기본 링크는 사용자가 제공한 티켓링크 모바일 주소를 반영했습니다. 구단별 예매 오픈 정책은 시즌, 멤버십 선예매, 이벤트 경기, 포스트시즌 여부에 따라 바뀔 수 있습니다.

정확도를 높이려면 실제 구단 공지에 맞춰 아래 값을 수정하세요.

```json
{
  "openOffsetDays": 7,
  "openTime": "11:00",
  "ticketUrl": "https://...",
  "certainty": "confirmed"
}
```

## 사용 방법

1. 앱을 엽니다.
2. 관심 경기의 날짜, 시간, 홈팀, 원정팀을 입력합니다.
3. 경기별 정확한 예매 URL을 알고 있다면 직접 입력합니다.
4. `캘린더 추가`를 눌러 휴대폰 캘린더에 30/10/5/1분 전 알림을 등록합니다.
5. 예매 오픈 전에는 예매 버튼이 잠겨 있습니다.
6. 예매 오픈 시각 이후 버튼이 `예매 바로가기`로 자동 활성화됩니다.
7. 예매처에서 사용자가 직접 좌석 선택과 결제를 진행합니다.

## 배포

정적 파일만 사용하는 PWA이므로 Vercel, GitHub Pages, Netlify 등에 바로 배포할 수 있습니다.

Vercel 기준으로는 별도 빌드 명령 없이 루트 디렉터리를 배포하면 됩니다.
