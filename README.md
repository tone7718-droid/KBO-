# 삼성 라이온즈 예매 보조 PWA

삼성 라이온즈 팬 기준으로 만든 **정시 예매 진입 보조 앱**입니다.

이 프로젝트는 대구 삼성라이온즈파크 홈경기의 예매 오픈 상태를 놓치지 않도록 돕는 정적 PWA입니다. 티켓링크 모바일 링크, 경기 시간, 예매 오픈 시각, 좌석 메모, 캘린더 알림, 브라우저 알림을 한 화면에서 관리합니다.

## 핵심 원칙

이 앱은 다음 기능을 의도적으로 제공하지 않습니다.

- CAPTCHA 우회
- 대기열 우회
- 좌석 자동 클릭
- 자동 결제
- 티켓링크 페이지 반복 새로고침
- 다계정/프록시 접속

목표는 예매처 시스템을 우회하는 것이 아니라, 사용자가 정시에 직접 예매할 수 있도록 준비 시간을 줄이는 것입니다.

## 현재 구현된 기능

- 삼성 홈경기 중심 UI
- 삼성 라이온즈 기본 예매 링크: `https://m.ticketlink.co.kr/sports/137/57`
- 2026.05.16 KIA @ 삼성 샘플 데이터 수정
  - 경기 시간: 17:00
  - 상태: 예매 가능으로 표시
- 잘못된 기존 로컬 샘플 데이터 초기화 버튼
- 인원 입력 제거
- 상대팀, 경기일, 시작 시간, 예매 오픈 시각, 좌석 메모만 입력
- 예매 오픈 전 버튼 잠금
- 예매 오픈 시각 이후 `예매 바로가기` 버튼 자동 활성화
- 앱 내부 1초 단위 카운트다운 갱신
- `.ics` 캘린더 알림 파일 생성: 30분 전, 10분 전, 5분 전, 1분 전
- 브라우저 로컬 알림 예약: 30분 전, 10분 전, 5분 전, 1분 전
- PWA manifest 및 service worker 구성
- 삼성 블루 중심 모바일 UI

## 중요한 제한 사항

현재 앱은 티켓링크의 좌석 현황을 직접 조회하지 않습니다. 예매 가능 여부는 앱에 저장된 예매 오픈 시각 기준으로 표시합니다.

`data/games.sample.json`의 2026.05.16 경기는 사용자의 정정 내용을 반영해 17:00 시작, 이미 예매 오픈된 상태로 등록했습니다.

## 사용 방법

1. 앱을 엽니다.
2. 잘못된 예전 데이터가 보이면 `데이터 초기화`를 누릅니다.
3. 삼성 홈경기의 날짜, 시간, 상대팀, 예매 오픈 시각을 입력합니다.
4. `캘린더 추가`를 눌러 휴대폰 캘린더에 30/10/5/1분 전 알림을 등록합니다.
5. 예매 오픈 시각 이후 버튼이 `예매 바로가기`로 자동 활성화됩니다.
6. 티켓링크에서 사용자가 직접 좌석 선택과 결제를 진행합니다.

## 배포

정적 파일만 사용하는 PWA이므로 Vercel, GitHub Pages, Netlify 등에 바로 배포할 수 있습니다.

Vercel 기준으로는 별도 빌드 명령 없이 루트 디렉터리를 배포하면 됩니다.
